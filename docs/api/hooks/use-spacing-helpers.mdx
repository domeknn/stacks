`useSpacingHelpers` returns two functions: `multiply` and `divide` allowing you to accordingly multiply or divide a value by the `spacing` value passed to `StacksProvider`.

### TypeScript

```typescript
type SpacingHelpers = {
  readonly multiply: (value: number) => number
  readonly divide: (value: number) => number
}

const useSpacingHelpers: () => SpacingHelpers
```

### ReScript

```typescript
type spacingHelpers = {
  multiply: (. float) => float,
  divide: (. float) => float,
}

let useSpacingHelpers: unit => spacingHelpers
```

### Example

```jsx live
function() {
  const DebugText = props => {
    const { marginBottom } = props
    const { multiply } = useSpacingHelpers()

    return (
      <View style={{ marginBottom: multiply(marginBottom), backgroundColor: 'rgba(97, 0, 255, .1)' }}>
        <Text>Debug</Text>
      </View>
    )
  }

  return (
    <StacksProvider spacing={4}>
      <DebugText marginBottom={2} />
      <DebugText marginBottom={8} />
      <DebugText />
    </StacksProvider>
  )
}
```
