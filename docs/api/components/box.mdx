`Box` is the most low-level layout component provided by `Stacks`.

`Box` accepts all [`View`](https://reactnative.dev/docs/view) props.

### Examples

```jsx live
function() {
  return (
    <StacksProvider>
      <Box direction="row" padding={2}>
        <Box flex="fluid">
          <Box flex="fluid" alignX="center" alignY="center">
            <Placeholder width={[160, 200, 240]} height={240} />
          </Box>
        </Box>
        <Box flex="1/3">
          <Box marginBottom={5}>
            <Stack space={1}>
              <Placeholder height={50} />
              <Placeholder height={50} />
              <Placeholder height={50} />
              <Placeholder height={50} />
            </Stack>
          </Box>
          <Box>
            <Tiles columns={2} space={1}>
              <Placeholder height={50} />
              <Placeholder height={50} />
              <Placeholder height={50} />
              <Placeholder height={50} />
            </Tiles>
          </Box>
        </Box>
      </Box>
    </StacksProvider>
  )
}
```
