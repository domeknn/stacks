`Inline` renders a set of components in a row with equal spacing around them, wrapping onto multiple lines when necessary. It also supports the horizontal alignment of its children.

`Inline` accepts all [`View`](https://reactnative.dev/docs/view) props.

### Examples

Simple usage.

```jsx live
function() {
  return (
    <StacksProvider>
      <Inline space={2}>
        <Placeholder width={90} />
        <Placeholder width={60} />
        <Placeholder width={120} />
        <Placeholder width={200} />
        <Placeholder width={100} />
        <Placeholder width={140} />
      </Inline>
    </StacksProvider>
  )
}
```

Multiple `Inline` components can be nested.

```jsx live
function() {
  return (
    <StacksProvider>
      <Inline space={10}>
        <Inline space={2}>
          <Placeholder width={60} />
          <Placeholder width={90} />
        </Inline>
        <Inline space={2}>
          <Placeholder width={60} />
          <Placeholder width={90} />
        </Inline>
      </Inline>
    </StacksProvider>
  )
}
```

Set horizontal alignment.

```jsx live
function() {
  return (
    <StacksProvider>
      <Inline space={2} align="center">
        <Placeholder width={90} />
        <Placeholder width={60} />
        <Placeholder width={120} />
        <Placeholder width={100} />
        <Placeholder width={100} />
      </Inline>
    </StacksProvider>
  )
}
```
