`Bleed` renders a container with negative margins allowing content to ["bleed"](https://en.wikipedia.org/wiki/Bleed_(printing)) into the surrounding layout.

### Space

```jsx live
const App = () => {
  return (
    <Stack space={4}>
      <Placeholder />
      <Bleed space={3}>
        <Placeholder />
      </Bleed>
      <Placeholder />
    </Stack>
  )
}

render(<App />)
```
