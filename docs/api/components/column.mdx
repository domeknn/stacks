Use the `Column` component along with `Columns`.

`Column` accepts all [`View`](https://reactnative.dev/docs/view) props.

### Examples

```jsx live
function() {
  return (
    <StacksProvider>
      <Columns alignX="between">
        <Column width="content">
          <Placeholder width={100} />
        </Column>
        <Column width="1/2">
          <Placeholder />
        </Column>
        <Column width="content">
          <Placeholder width={100} />
        </Column>
      </Columns>
    </StacksProvider>
  )
}
```

Use the `height` property to adjust the column height against other columns.

```jsx live
function() {
  return (
    <StacksProvider>
      <Columns space={4}>
        <Column width="content" height="fluid">
          <Rows space={4}>
            <Row height="content">
              <Placeholder width={200} height={100} />
            </Row>
            <Row>
              <FluidPlaceholder width={200} />
            </Row>
          </Rows>
        </Column>
        <Column>
          <Placeholder height={300} />
        </Column>
      </Columns>
    </StacksProvider>
  )
}

```
