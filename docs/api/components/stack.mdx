`Stack` distributes white space evenly between components (vertically or horizontally if you set the `horizontal` prop to `true`).

`Stack` accepts all [`View`](https://reactnative.dev/docs/view) props.

### Examples

Set spacing between elements stacked vertically.

```jsx live
function() {
  return (
    <StacksProvider>
      <Stack space={1}>
        <Placeholder />
        <Placeholder />
        <Placeholder />
      </Stack>
    </StacksProvider>
  )
}
```

Multiple `Stack` components can be nested.

```jsx live
function() {
  return (
    <StacksProvider>
      <Stack space={10}>
        <Stack space={2}>
          <Placeholder />
          <Placeholder />
        </Stack>
        <Stack space={2}>
          <Placeholder />
          <Placeholder />
          <Placeholder />
        </Stack>
      </Stack>
    </StacksProvider>
  )
}
```

Set horizontal alignment.

```jsx live
function() {
  return (
    <StacksProvider>
      <Stack space={1} align="center">
        <Placeholder width={240} />
        <Placeholder width={300} />
        <Placeholder width={240} />
      </Stack>
    </StacksProvider>
  )
}
```

Stack elements horizontally.

```jsx live
function() {
  return (
    <StacksProvider>
      <Stack space={1} horizontal={true} align="center">
        <Placeholder width={100} height={140} />
        <Placeholder width={100} height={200} />
        <Placeholder width={100} height={140} />
      </Stack>
    </StacksProvider>
  )
}
```

Stack with a custom divider.

```jsx live
function() {
  return (
    <StacksProvider>
      <Stack space={4} divider={<Divider />}>
        <Placeholder height={100} />
        <Placeholder height={140} />
        <Placeholder height={80} />
      </Stack>
    </StacksProvider>
  )
}
```
