It returns the `Stacks` context object.

<Tabs {...defaultTabsProps}>

declare type Breakpoint = [string, number]

</TabItem>

## Example

```jsx live
function() {
  const DebugText = () => {
    const context = useStacks()

    return (
      <Text>{JSON.stringify(context)}</Text>
    )
  }

  return (
    <StacksProvider spacing={4}>
      <DebugText />
    </StacksProvider>
  )
}
```
